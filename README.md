# Capputeeno
#### Projeto front-end inspirados em e-commerces, esse no caso é de camisetas e canecas, no projeto é possivel se cadastrar, realizar login, ver os produtos de diferentes categorias, detalhes dos produtos, adicionar produto ao carrinho e finalizar pedido, o projeto é baseado e um array e manipula os dados através do localstorage não é consumida nenhuma api.

#### Teconologias: Javascript, React.js.
#### Link: https://capputeeno-lucas-quevedo.surge.sh/
